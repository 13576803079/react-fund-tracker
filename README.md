# React Fund Tracker

A React-based fund tracking application similar to the original Vue funds extension. This application allows you to monitor your fund investments with real-time data, track gains/losses, and manage your portfolio efficiently.

## Features

- View real-time fund data with estimated net value and growth rates
- Track your investment gains and losses
- Manage multiple funds in one place
- Support for dark/light mode
- Customizable display options (show/hide columns)
- Portfolio summary with total gains/losses
- Search and add new funds easily
- Responsive design for desktop use

## Installation

1. Clone the repository:
```bash
git clone https://github.com/[YOUR_USERNAME]/react-fund-tracker.git
```

2. Install dependencies:
```bash
cd react-fund-tracker
npm install
```

3. Start the development server:
```bash
npm start
```

## Configuration

The application uses fund data from East Money API. The original Vue extension was built with similar functionality for Chrome Extension use. This React version provides a standalone desktop application alternative.

## Technologies Used

- React 18
- Material UI
- Axios for API calls
- CSS Modules for styling

## Development

This project was bootstrapped with Create React App. For development, simply run:

```bash
npm start
```

The application will open in your default browser at http://localhost:3000

## Building for Production

To create an optimized production build:

```bash
npm run build
```
